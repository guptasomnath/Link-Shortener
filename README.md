# Full Stack Link-Shortener
This is a full-stack responsive server-side **link shortener** web app 

# Techonologys Used
Node Js And Express Js For Backend Development,
##
EJS for view engine,
##
MongoDB for database,
##
Deployed At Vercel Cloud
